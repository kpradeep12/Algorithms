## Problems

### 1. In Order Traversal
Traverse the left node (node.left)  
Visit the node  
Traverse the right node (node.right)

### 2. Pre Order Traversal
Visit the node  
Traverse the left node (node.left)  
Traverse the right node (node.right)

### 3. Post Order Traversal
Traverse the left node (node.left)  
Traverse the right node (node.right)
Visit the node
